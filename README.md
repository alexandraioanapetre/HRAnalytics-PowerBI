👥 HR Analytics - SQL Server & Power BI Dashboard

📌 About the project

A complete **HR Analytics** project for a fictional tech company, including a relational database in SQL Server and an interactive Power BI dashboard covering headcount, salaries, performance, and employee turnover.

📊 Power BI Dashboard

**Page 1 — Overview**
- KPI Cards: Total Employees, Active Employees, Turnover Rate %, Average Salary, Avg Performance Score
- Employees by department
- Seniority distribution

**Page 2 — Salaries & Performance**
- Average salary by department
- Average salary by seniority level
- Salary vs performance correlation (scatter plot)
- Filters by gender and department

**Page 3 — Turnover & Training**
- Resignation reasons breakdown
- Performance category distribution
- Resigned employees detail table
- Resignations by department

📝 SQL Exercises

This repo also includes SQL practice exercises covering:
- SELECT, WHERE, ORDER BY
- GROUP BY, HAVING
- INNER JOIN, LEFT JOIN (multiple tables)
- Subqueries
- CASE WHEN

🛠️ Technologies used

- **SQL Server 2022** — relational database
- **SSMS** — SQL Server Management Studio
- **Power BI Desktop** — visualizations and dashboard
- **DAX** — calculated measures (Turnover Rate, Avg Salary, Avg Performance)

📈 Dashboard insights

- **Engineering** is the largest department (6 employees)
- **Directors** earn on average 3x more than **Junior** employees
- **Better Opportunity** is the main reason employees leave
- Most employees score **Exceeds Expectations** in performance reviews
